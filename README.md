# VLSI Chip Routing Visualizer

Interactive tool for visualizing and computing shortest paths (Dijkstra's algorithm) between gates on a VLSI chip layout. Features both Streamlit and Flask+HTML interfaces, with Plotly visualization and NetworkX routing logic. Ideal for VLSI/EDA education and simulation.

## Features
- Select start and end gates interactively
- Visualize chip graph and shortest routing path
- Streamlit and Flask interface options

## Folder Structure
- `streamlit_app/` — Streamlit implementation
- `flask_app/` — Flask web app, with HTML interface

## Setup & Run
1. `git clone https://github.com/MohamedNavith/chip-routing-visualizer.git`
2. See the directory `streamlit_app` or `flask_app` for specific instructions
